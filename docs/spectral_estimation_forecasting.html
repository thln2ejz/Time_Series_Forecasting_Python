Smoothing with a moving average filter¶

A pandas data frame allows computing a moving average with a specified window size. To test the effect of applying such a filter, run the below code. What are the filter coefficients used for the 7-point moving average filter? Note the number of NaN values at the start and end of each result when applying a 3-point and 7-point filter. By default, the length of the filter should be odd, but in case it is even, more of the filter is placed backward in time than forward.

Note: alternatively, we can use convolve() from scipy.signal to convolve a time series with specified filter coefficients as shown below.

import matplotlib.pyplot as plt

#alternatively: scipy

b = (np.ones(3))/3 
y = signal.convolve(x, b, mode='valid') #NaNs are automatically removed

Detrending & deseasonalizing by differencing¶

When both trend and seasonality are present, differencing can be used to remove both. The time series of monthly accidental deaths in the US contains a seasonal component of period 12. Read in and plot this time series. To remove seasonality, we may use the $\nabla_{12}$ operator which by definition gives $Y_t = \nabla_{12}X_t = X_t - X_{t-12}$. Apply this operator on the given time series using the pandas' diff() while specifying the required lag (i.e. periods argument). Show the plot as in Figure 1-26 of the textbook.

Note: another way to apply differencing is subtracting a forward shifted version of the time series (pandas' shift(periods=...)) from the original time series.

To remove trend in the resulting (deseasonalized) series, apply $\nabla Y_t = Y_t - Y_{t-1}$ whereby the equivalent final operation on $X_t$ is given by $\nabla \nabla_{12} X_t$. Plot the result as in Figure 1-27 of the textbook.

Forecasting¶

Once model parameters have been estimated, predictions can be done using forecast() as shown below, say for a horizon of 2 samples ahead.

In [16]:
model = ArmaProcess([1], [1, 0.8])      #MA(1)
x = model.generate_sample(nsample=100)  #simulation

model = ARIMA(x, order=(0, 0, 1)).fit() #estimation
y_pred = model.forecast(steps=2) 
 
#alternatively: same result
start = len(x)
end = len(x) + 1 
y_pred = model.predict(start=start, end=end, typ='levels')
