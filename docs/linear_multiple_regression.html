MA(1) process¶

Recall that the MA(1)-process defined by

$$X_t = Z_t + \theta Z_{t-1}$$

has the following ACVF:

$$\gamma_X(t+h,t) = \begin{cases} \sigma^2(1+ \theta^2), & \text{if $h=0$} \\ \sigma^2 \theta, & \text{if $h=\pm1$} \\ 0, & \text{if $|h|>1$} \\ \end{cases}$$

Therefore, its ACF is given by:

$$\rho_X(h) = \begin{cases} 1, & \text{if $h=0$} \\ \theta/(1+\theta^2), & \text{if $h=\pm1$} \\ 0, & \text{if $|h|>1$} \\ \end{cases}$$

Suppose we have an MA(1)-process with $\theta = 0.8$. Create an array of the theoretical ACF values $\rho(0)$ through $\rho(20)$, and plot these as a stem plot.

Simulate an MA(1) time series with $\theta=0.8$ using ArmaProcess() (module: statsmodels.tsa.arima_process) as shown below. The simulation utilizes the MA(1)-equation. AR(1) process¶

Recall that the AR(1)-process defined by

$$X_t = \phi X_{t-1} + Z_t$$

has the following ACVF:

$$\gamma_X(h) = \dfrac{\sigma^2}{1-\phi^2} \phi^{|h|}$$

Therefore, its ACF is given by:

$$\rho_X(h) = \phi^{|h|}$$

Suppose we have an AR(1)-process with $\phi = -0.5$. Create an array of the theoretical ACF values $\rho(0)$ through $\rho(20)$, and plot these as a stem plot.

Simulate an AR(1) time series with $\phi=-0.5$ (again using ArmaProcess()). Plot the simulated time series; also plot the sample ACF of this time series, and add the theoretical values highlighted as points.
