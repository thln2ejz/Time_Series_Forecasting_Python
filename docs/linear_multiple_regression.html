Linear regression¶

Linear regression with one independent variable assumes the model

$$y_i = \beta_0 + \beta_1 x_i + \epsilon_i$$

where $\beta_0$ and $\beta_1$ are the regression coefficients.

Diagnostics¶

To assess the model, we need to study the residuals $y_i - \widehat{y}_i$. There are a number of diagnostic checks we may perform. Here are some:

  1. Plot the residuals; is there any apparent pattern or trend? If so, then this would not be a good indication, meaning that our linear model is inaccurate. The residuals are expected to be normally distributed with zero mean and constant variance.
  1. To (roughly) check the normality assumption, plot the histogram of the residuals; does it resemble a normal distribution?
  1. Furthermore, we may check the normal QQ-plot (i.e. quantile-quantile plot) to see if the actual quantiles follow the theoretical quantiles of the normal distirbution. A QQ-plot in this case can be obtained using scipy's stats.probplot() (or statsmodels' qqplot()). Do the points follow the QQ-line (i.e. y=x line)? If so, then this gives evidence of a normal distribution.
  1. Finally, we may formally test for normality using the Shapiro-Wilk test (hint: scipy.stats.shapiro()). Is the p-value low enough to reject the null hypothesis of normality?

Identifying outliers¶

In the context of least-squares regression, we may also examine Cook's distance for each data point. This distance captures the influence of individual points on the resulting regression model by computing the variation in the fitted response when a single data point under consideration is removed from the dataset versus when it is included. This can help us identify outliers which would have a comparatively large influence (Cook's distance) compared to other data points. Compute Cook's distance for each data point and create a plot of these values. Note: Cook's distance can be obtained from a model trained by statsmodels' ols() by using the method get_influence() and summary_frame() (Cook's distance will be one of the columns in the data frame).
