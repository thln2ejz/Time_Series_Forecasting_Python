def synth_data(n_samples, sigma):
    X = np.random.uniform(low=0,high=2,size=n_samples)
    X = np.sort(X)
    y = np.sin(4*X) + np.random.normal(loc=0,scale=sigma,size=n_samples) 
    return X, y

We will use a radial basis function (RBF) as kernel for constructing the covariance matrix. The RBF kernel is given by:

$$k(x_1,x_2) = exp\left( \frac{-1}{2}||x_1 - x_2||^2/l^2 \right)$$

Using this kernel, we build a covariance matrix.

In [3]:
def kernel_rbf(x1, x2, l):
    Sigma = [[np.exp(-0.5*abs((x1[i]-x2[j])/l)**2) for j in range(len(x2))] for i in range(len(x1))]
    Sigma = np.array(Sigma) 
    return Sigma

Exercise 3. So far we have only sampled from the prior distribution. Let us now consider the artificially generated dataset in Exercise 1. The hyperparameters for setting up a Gaussian process in our case are the characteristic length-scale of the RBF kernel, $l$, and the standard deviation of Gaussian noise, $\sigma_{n}$ which we can tolerate around the training objects. Let us specify these hyperparameters arbitrarily for now with $\sigma_{n} = 0.5$ and $l=0.1$.

Recall that the test outputs, $f_{*}$, are given by:

$$\textbf{f}_{*} \sim \mathcal{N}\left( K(X_{*},X) [K(X,X) + \sigma_n^2 \textbf{I}]^{-1} \textbf{y}, K(X_{*},X_{*})-K(X_{*},X)[K(X,X)+\sigma_n^2 \textbf{I}]^{-1}K(X,X_{*})\right) $$

Start by computing the kernels:
